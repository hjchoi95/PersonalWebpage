(static) accessAccount(username, password, callback)

This function retrieves the ID of the account with the specified username and password passed into the function.

This function retrieves the ID of the account with the specified username and password passed into the function. The "status" field of the result object will be set to "SUCCESS" if the function succeeds, otherwise it will be set the "FAIL: " followed by an error message. The "result" field will be an object with the fields username, password_hash, customer_id, and msg_user_id. The msg_user_id field is used for messaging system management.

(static) accessProfile(username, callback)

Acceess profile data from db given username.

This function queries the database for a profile with the given username, and returns all the data of the profile as an object. The "status" field of the result object will be set to "SUCCESS" if the function succeeds, and the result field will be set to the object containing all the data of the profile. Otherwise, the "status" field will be set to "FAIL: " followed by an appropriate error message, and the "result" field will be set to null.

(static) createAccount(accountInfo, callback)

Creates an account and profile in the db given username and password.

This function creates an account and profile with the username, password, and profile data specified in the accountInfo object and inserts it into the database. The "status" field of the result object will be set to "SUCCESS" if the function succeeds, therwise it will be set the "FAIL: " followed by an error message. (static) modifyAccount(username, password, fieldType, newField, callback)

Modifies account data in db given username and password.

This function updates account password or ban status information in the database. The "status" field of the result object will be set to "SUCCESS" if the function succeeds, otherwise it will be set the "FAIL: " followed by an error message. The "result" field is always set to null. Note that this function cannot modify the username of an account.

Parameters:
Name Type Description
username string

The username of the account to modify.
password string

The password of the account to modify.
fieldType string

The field that the function will modify. Valid values are "password" and "ban".
newField string

The new value of the field that will be modified. For bans, valid values of newField are restricted to: "none", "oneweek", "twoweeks", "onemonth", "halfyear", "year", and "perma".
callback responseCallback

The callback that handles the response
