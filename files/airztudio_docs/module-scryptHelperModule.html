(static) updatePwdHashQuery(db, table, username, password, passwordHash) → {Promise}

Updates password_hash in database.

This function rehashes the current password and updates the password hash stored in the database with it. This is useful if the scrypt parameters in the website change.

The promise object will be resolved with "SUCCESS" if the password hash does not require a re-hash, or if the password hash has been successfully updated. The object will be rejected with an error message if the hash function fails or the password hash cannot be updated in the database for some reason.
