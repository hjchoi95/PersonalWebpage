(static) createPackage(studioId, packageContents, packagePhoto, callback)

Inserts the data in packageContents into the database.

Inserts a packageContents JSON object into the database. The function returns a JSON object in the callback, which has fields {"status", "result"}. "status" is set to "SUCCESS" if the insertion is successful, otherwise it will be set to "FAIL: " along with an error message. "result" will be set to an object containing the ID of the package in the database and an array of all the inserted photograph IDs if the insertion succeeds, and null if it does not.

(static) modifyPackage(studioId, packageId, packageContents, callback)

Modifies a package by studio, package id, and packageContents.

Modifies a package in the database by replacing the data fields in the row with the data in the packageContents JSON object. This function cannot modify any photographs that may be linked to the package. The function returns a JSON object in the callback, which has fields {"status", "result"}. "status" is set to "SUCCESS" if the modification is successful, otherwise it will be set to "FAIL: " along with an error message. "result" is always set to null.
